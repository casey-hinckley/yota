How Rankings Work
Skip Scores

Athletes earn +3 points for full practice attendance with zero skips. Skips result in -1 point each. Partial attendance (leaving early/arriving late) or absences earn 0 points but skips still count as penalties.

Skip Streaks

Tracks the current number of consecutive days an athlete has had zero skips. One skip breaks the streak back to zero.

Goal Streaks

Tracks the current number of consecutive days an athlete has achieved their daily goals. Missing a goal breaks the streak.
